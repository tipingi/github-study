1. 소스트리(Source Tree) 설치 - VS Code 사용
소스트리 = git 사용을 도와주는 GUI 프로그램.

sourcetree, bitbucket: 아틀라시안이라는 호주 회사에서 만든 Git 호스팅 사이트.
mercurial: git 버전관리 툴


2. 구성
로컬저장소 목록: 컴퓨터에 저장되 어있는 Git 로컬 저장소 목록.
원격저장소 목록: 계정에 연결되어 있는 모든 원격 저장소 목록
원격저장소 클론(Clone): 원격 서버에 올라와 있는 Git 저장소 -> 로컬 컴퓨터에 다운로드 및 연동 가능
로컬저장소 추가: 내 컴퓨터에만들어져 있는 로컬저장소를 소스트리에서 관리 가능
로컬저장소 생성: 컴퓨터에 있는 일반 폴더를 Git으로 버젼 관리 할 수 있도록 로컬 저장소  생성.


3. 비쥬얼 스튜디오 코드.






4. 로컬저장소를 소스트리에 불러오기
소스트리 실행 -> [Add]
Clone: 원격저장소를  내 컴퓨터에 받아오고 소스트리에도 추가
Add: 내 컴퓨터에서 이미 만든 로컬저장소를 소스트리에 추가
Create: 내 컴퓨
